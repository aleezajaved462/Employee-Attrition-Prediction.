# Employee Attrition Prediction Pipeline

An end-to-end Machine Learning pipeline to predict employee attrition risk and identify key workforce drivers using the IBM HR Analytics dataset. Developed as part of the Hexovate ML Internship Assignment.

##  Project Overview
Employee attrition is a costly challenge for any organization. This project builds a classification model to proactively identify employees at risk of leaving, enabling data-driven HR retention strategies.

##  Repository Structure
- `data/`: Contains the HR analytics dataset.
- `notebooks/`: Contains the complete Jupyter Notebook with EDA, preprocessing, and model training.
- `requirements.txt`: List of dependencies required to run the project.
